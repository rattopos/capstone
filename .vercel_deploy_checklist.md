# Vercel 배포 체크리스트

배포 전 확인사항:

## ✅ 필수 확인사항

- [ ] `requirements.txt`에 모든 의존성이 포함되어 있는지 확인
- [ ] 로컬에서 `python app.py`로 정상 실행되는지 확인
- [ ] Git 저장소에 커밋 및 푸시 완료
- [ ] Vercel 계정 생성 및 로그인 완료

## 📦 배포 단계

1. **Vercel CLI 설치** (선택사항):
   ```bash
   npm i -g vercel
   ```

2. **프로젝트 디렉토리에서 배포**:
   ```bash
   vercel login
   vercel
   ```

3. **프로덕션 배포**:
   ```bash
   vercel --prod
   ```

## 🔍 배포 후 확인

- [ ] 대시보드 페이지 접속 확인 (`/`)
- [ ] 파일 업로드 기능 테스트
- [ ] 보도자료 미리보기 기능 테스트
- [ ] 에러 로그 확인 (Vercel 대시보드)

## ⚠️ 주의사항

- Vercel 서버리스 환경에서는 업로드된 파일이 임시 저장소(`/tmp`)에만 저장됩니다
- 파일은 함수 실행 후 삭제되므로, 업로드 후 즉시 처리해야 합니다
- 대용량 Excel 파일 처리 시 타임아웃이 발생할 수 있습니다
