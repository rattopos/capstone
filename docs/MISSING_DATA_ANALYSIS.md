### 1. 서비스업생산
- **전국 지수**: `get_nationwide_data()` 함수에서 `production_index` 추출 실패
- **지역별 지수**: 요약 박스 생성 시 지역별 지수 추출 실패
- **지역별 세부업종**: `get_regional_data()` 함수에서 `top_industries` 추출 실패

### 2. 수출/수입
- **전국 데이터**: `generate_report_data()` 함수에서 전국 데이터 추출 실패
- **테이블 데이터**: `_generate_summary_table_from_aggregation()` 함수에서 데이터 추출 실패
- **원인 가능성**: 
  - 엑셀 파일의 시트 구조 불일치
  - 동적 컬럼 감지 실패
  - 데이터가 다른 시트에 있음

### 3. 물가
- **전국 소비자물가**: 전국 데이터 추출 실패
- **품목 데이터**: 품목별 데이터 추출 실패

## 확인 필요 사항

   - `src/generators/service_industry_generator.py`의 `get_regional_data()`
