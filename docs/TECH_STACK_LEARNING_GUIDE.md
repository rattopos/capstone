## 프로젝트 기술 스택 개요

### Backend
- **Flask 2.3+**: Python 웹 프레임워크
- **Werkzeug**: WSGI 유틸리티 라이브러리
- **Jinja2**: 템플릿 엔진
- **Python 3.10+**: 프로그래밍 언어

### 데이터 처리
- **pandas 2.0+**: 데이터 분석 및 조작
- **numpy 1.24+**: 수치 계산
- **openpyxl 3.1+**: Excel 파일 처리
- **xlwings**: Excel 애플리케이션 연동
- **formulas**: Excel 수식 계산

### 파일 처리
- **BeautifulSoup4**: HTML/XML 파싱
- **Pillow**: 이미지 처리
- **lxml**: XML/HTML 처리 (HWPX 파일)

### Frontend
- **HTML5/CSS3**: 마크업 및 스타일링
- **JavaScript (Vanilla)**: 클라이언트 사이드 로직
- **Chart.js**: 데이터 시각화
- **Jinja2 템플릿**: 서버 사이드 렌더링

### 데스크톱 애플리케이션
- **PyQt6**: GUI 프레임워크
- **PyQt6-WebEngine**: 웹뷰 통합

### 배포 및 인프라
- **Vercel**: Serverless 배포 플랫폼
- **Python-dotenv**: 환경 변수 관리
- **requests**: HTTP 클라이언트

### 기타
- **python-dateutil**: 날짜/시간 처리
- **pytz**: 타임존 처리
- **PyInstaller**: 실행 파일 패키징

## 학습 로드맵

### Phase 1: 기초 다지기 (1-2개월)
**목표**: 각 기술의 기본 개념과 사용법 익히기

1. **Python 기초 심화**
   - 객체지향 프로그래밍 (OOP)
   - 데코레이터, 제너레이터, 컨텍스트 매니저
   - 비동기 프로그래밍 (asyncio)
   - 타입 힌팅 및 mypy

2. **Flask 기초**
   - Flask 애플리케이션 구조
   - 라우팅 및 요청 처리
   - Blueprint 패턴
   - 세션 및 쿠키 관리

3. **데이터 처리 기초**
   - pandas 기본 연산
   - numpy 배열 조작
   - Excel 파일 읽기/쓰기

### Phase 2: 중급 실전 (2-3개월)
**목표**: 실제 프로젝트에서 사용되는 패턴과 고급 기능 학습

1. **Flask 고급**
   - RESTful API 설계
   - 에러 핸들링 및 로깅
   - 파일 업로드/다운로드
   - 인증 및 권한 관리
   - 데이터베이스 연동 (SQLAlchemy)

2. **데이터 처리 고급**
   - pandas 고급 인덱싱 및 그룹화
   - 데이터 정제 및 변환
   - 성능 최적화
   - 대용량 데이터 처리

3. **프론트엔드 기초**
   - HTML5 시맨틱 마크업
   - CSS3 레이아웃 (Flexbox, Grid)
   - JavaScript ES6+ 문법
   - DOM 조작 및 이벤트 처리

### Phase 3: 고급 심화 (3-4개월)
**목표**: 전문가 수준의 기술과 아키텍처 이해

1. **웹 애플리케이션 아키텍처**
   - MVC/MVP 패턴
   - 의존성 주입
   - 테스트 주도 개발 (TDD)
   - 코드 리팩토링

2. **성능 최적화**
   - 쿼리 최적화
   - 캐싱 전략
   - 비동기 처리
   - 프로파일링 및 디버깅

3. **배포 및 DevOps**
   - Docker 컨테이너화
   - CI/CD 파이프라인
   - 클라우드 서비스 이해
   - 모니터링 및 로깅

### Phase 4: 전문가 수준 (4-6개월)
**목표**: 실무에서 요구되는 고급 기술과 문제 해결 능력

1. **시스템 설계**
   - 확장 가능한 아키텍처
   - 마이크로서비스 패턴
   - 메시지 큐 및 이벤트 기반 아키텍처

2. **보안**
   - 웹 보안 취약점 (OWASP Top 10)
   - 인증/인가 구현
   - 데이터 암호화

3. **특수 기술**
   - HWPX 파일 포맷 이해 및 처리
   - 복잡한 Excel 수식 처리
   - 한글 인코딩 처리
