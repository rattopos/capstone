# 📚 프로젝트 용어 사전

> 지역경제동향 보도자료 자동 생성 시스템에서 사용되는 용어들을 정리한 사전입니다.

---

## 1. 프로젝트 핵심 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **규칙기반 시스템** | 사전에 정의된 규칙에 따라 데이터를 처리하고 문장을 생성하는 시스템. LLM과 달리 100% 정확한 출력을 보장한다. | 프로젝트 핵심 아키텍처 |
| **Generator** (생성기) | 엑셀 파일에서 보도자료에 필요한 데이터를 추출하는 Python 모듈. `templates/*_generator.py` 파일로 구현된다. | 프로젝트 핵심 컴포넌트 |
| **Schema** (스키마) | 보도자료에서 사용할 데이터의 구조와 위치를 정의한 JSON 파일. `templates/*_schema.json` 파일로 구현된다. | 프로젝트 핵심 컴포넌트 |
| **Template** (템플릿) | 보도자료의 HTML 형식을 정의한 Jinja2 템플릿 파일. `templates/*_template.html` 파일로 구현된다. | 프로젝트 핵심 컴포넌트 |
| **3계층 구조** | Generator → Schema → Template로 구성된 보도자료 생성 아키텍처. 데이터 추출, 구조 정의, 형식 정의를 분리한다. | 프로젝트 핵심 아키텍처 |
| **이중 프로세스 구조** | 프로세스 1: 기초자료 수집표 → 분석표 자동 생성 / 프로세스 2: 분석표 + GRDP → 보도자료 HTML 생성 | 프로젝트 핵심 프로세스 |
| **가중치 기반 순위 시스템** | 17개 시도의 경제지표 증감률에 가중치를 적용하여 종합 순위를 자동 계산하는 시스템. | 프로젝트 핵심 기능 |
| **한글 복붙용 HTML** | 한글(HWP) 프로그램에 복사-붙여넣기하기 좋게 최적화된 HTML 형식. 표와 텍스트가 그대로 유지된다. | 프로젝트 출력 포맷 |
| **결측치 처리** | 데이터가 없거나 비공개인 경우 노란색 형광펜으로 하이라이트하고, 사용자가 직접 수정할 수 있도록 하는 기능. | 프로젝트 핵심 기능 |

## 2. 데이터 파일 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **기초자료 수집표** | KOSIS 등에서 수집한 원본 통계 데이터를 담은 엑셀 파일. 17개 시트(광공업생산, 서비스업생산, 소비, 고용률, 실업자 수, 지출목적별 물가, 품목성질별 물가, 건설, 수출, 수입, 시도 간 이동, 분기 GRDP 등)로 구성된다. | 프로젝트 입력 데이터 |
| **분석표** | 기초자료를 가공하여 집계 및 분석 수식이 적용된 엑셀 파일. 42개 이상의 시트로 구성되며, 보도자료 생성의 입력 자료이다. 각 지표마다 집계 시트, 분석 시트, 참고 시트로 구성된다. | 프로젝트 입력 데이터 |
| **집계 시트** | 분석표에서 기초자료의 원본 값을 저장한 시트. 시트명에 '집계'가 포함된다. 예: `A(광공업생산)집계`, `B(서비스업생산)집계`, `C(소비)집계` | 분석표 구성 요소 |
| **분석 시트** | 분석표에서 집계 시트의 데이터를 참조하여 증감률, 기여도, 순위 등을 계산하는 수식이 포함된 시트. 시트명에 '분석'이 포함된다. 예: `A 분석`, `B 분석`, `C 분석` | 분석표 구성 요소 |
| **참고 시트** | 분석표에서 추가 참고 정보를 담은 시트. 일부 지표에만 존재한다. 예: `A 참고`, `B 참고`, `F'참고` | 분석표 구성 요소 |
| **GRDP 시트** | 분석표에 추가되는 지역내총생산(GRDP) 데이터를 담은 시트. 기초자료 변환 과정에서 자동으로 추가된다. | 분석표 구성 요소 |
| **이용관련 시트** | 분석표에서 연도/분기 설정 정보를 담은 시트. K17 셀에 연도, M17 셀에 분기가 저장된다. | 분석표 구성 요소 |

## 3. 보도자료 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **지역경제동향** | 17개 시·도별 경제 현황을 분석한 국가데이터처의 분기별 보도자료. 생산, 소비, 물가, 고용 등을 포함하며, 총 50개 항목(78페이지)으로 구성된다. | 프로젝트 목표 산출물 |
| **요약 보도자료** | 표지, 목차, 인포그래픽, 요약 페이지들로 구성된 보도자료. 총 9개 페이지. | 보도자료 카테고리 |
| **부문별 보도자료** | 경제 부문(생산, 소비, 고용 등)별로 전국 17개 시·도의 현황을 분석한 보도자료. 총 10개 페이지. | 보도자료 카테고리 |
| **시도별 보도자료** | 특정 시·도의 모든 경제 지표(생산, 소비, 고용 등)를 종합 분석한 보도자료. 17개 시도 + 참고_GRDP로 총 18개 페이지. | 보도자료 카테고리 |
| **통계표** | 숫자 데이터를 행과 열로 정리한 표. 보도자료 부록에 세부 수치를 기록한다. 총 13개 페이지. | 보도자료 카테고리 |
| **인포그래픽** | 정보(Information)와 그래픽(Graphic)의 합성어. 복잡한 데이터를 시각적으로 표현한 이미지. 대시보드에서만 표시되며 한글 문서로 내보내지 않는다. | 보도자료 구성 요소 |
| **일러두기** | 보도자료 본문 앞에 위치하여 통계의 작성 목적, 용어 정의, 주의사항 등을 안내하는 페이지. | 보도자료 구성 요소 |

## 4. 경제/통계 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **광공업생산지수** (IIP) | 광업과 제조업의 생산 활동 수준을 나타내는 지수. 기준 연도(2020년=100)를 100으로 설정하고 생산량의 변화를 측정한다. | [한국은행 경제용어](https://www.bok.or.kr/portal/ecEdu/ecWordDicary/search.do?menuNo=200688) |
| **서비스업생산지수** | 서비스업 분야의 생산 활동 수준을 측정하는 지수. 도소매, 운수, 숙박음식, 정보통신 등의 서비스업 생산량 변화를 나타낸다. | [e-나라지표](https://www.index.go.kr/unity/potal/indicator/IndexInfo.do?clasCd=10&idxCd=F0053) |
| **소매판매액지수** | 소매업체의 판매액 변화를 측정하는 지수. 소비 동향을 파악하는 대표적인 지표로, 가계의 소비 지출 수준을 반영한다. | [국가통계포털 KOSIS](https://kosis.kr/statHtml/statHtml.do?orgId=101&tblId=DT_1KE1001) |
| **GRDP** (지역내총생산) | Gross Regional Domestic Product의 약자. 일정 기간 동안 특정 지역 내에서 생산된 모든 최종 재화와 서비스의 시장 가치 합계. GDP의 지역 버전이다. | [위키백과 - 지역내총생산](https://ko.wikipedia.org/wiki/지역내총생산) |
| **고용률** | 15세 이상 생산가능인구 중 취업자가 차지하는 비율. 수식: (취업자 수 / 생산가능인구) × 100 | [위키백과 - 고용률](https://ko.wikipedia.org/wiki/고용률) |
| **실업률** | 경제활동인구 중 실업자가 차지하는 비율. 수식: (실업자 수 / 경제활동인구) × 100 | [위키백과 - 실업률](https://ko.wikipedia.org/wiki/실업률) |
| **소비자물가지수** (CPI) | Consumer Price Index의 약자. 가계가 소비하는 상품과 서비스의 가격 변동을 측정하는 지수. 인플레이션 측정의 대표적 지표이다. | [위키백과 - 소비자물가지수](https://ko.wikipedia.org/wiki/소비자_물가_지수) |
| **건설수주액** | 건설업체가 발주자로부터 수주한 공사 금액의 합계. 건설업 경기 전망을 나타내는 선행지표로 활용된다. | [e-나라지표](https://www.index.go.kr/unity/potal/indicator/IndexInfo.do?clasCd=10&idxCd=4201) |
| **수출액** | 국내에서 생산된 재화가 국경을 넘어 외국으로 판매된 금액의 합계. 대외 무역 수지를 파악하는 핵심 지표이다. | [한국무역협회](https://stat.kita.net/) |
| **수입액** | 외국에서 생산된 재화가 국내로 반입된 금액의 합계. 내수 경기 및 산업 생산에 대한 의존도를 파악할 수 있다. | [한국무역협회](https://stat.kita.net/) |
| **국내인구이동** | 국내 시·도 간 주민등록 주소지 이동을 의미. 순이동자수는 전입자에서 전출자를 뺀 값이다. | [국가통계포털 KOSIS](https://kosis.kr/statHtml/statHtml.do?orgId=101&tblId=DT_1B26001_A01) |
| **전년동기비** (YoY) | Year over Year의 약자. 당해 분기를 전년 같은 분기와 비교한 것. 예: 2025년 2분기 vs 2024년 2분기 | [Investopedia - YoY](https://www.investopedia.com/terms/y/year-over-year.asp) |
| **전분기비** (QoQ) | Quarter over Quarter의 약자. 당해 분기를 직전 분기와 비교한 것. 예: 2025년 2분기 vs 2025년 1분기 | [Investopedia - QoQ](https://www.investopedia.com/terms/q/quarter-over-quarter.asp) |
| **증감률** | 두 시점 간의 값 변화를 백분율로 나타낸 것. 수식: ((당기값 - 기준값) / 기준값) × 100 | [위키백과 - 변화율](https://ko.wikipedia.org/wiki/변화율) |
| **기여도** | 전체 성장률에서 특정 항목이 기여한 정도를 나타내는 값. 수식: ((당기값 - 전기값) / 전체 전기값) × 100 | [한국은행 경제통계시스템](https://ecos.bok.or.kr/) |
| **지수** (Index) | 기준 시점의 값을 100으로 설정하고, 비교 시점의 값을 상대적인 비율로 나타낸 수치. 시계열 비교에 유용하다. | [위키백과 - 경제지수](https://ko.wikipedia.org/wiki/경제_지수) |
| **가중치** | 각 업종이나 지표의 중요도를 나타내는 수치. 증감률에 가중치를 곱하여 종합 순위를 계산할 때 사용된다. 일부 업종은 비공개(0)로 설정될 수 있다. | 프로젝트 핵심 개념 |
| **기여율** | 전체 성장률에서 특정 업종이나 항목이 기여한 비율. 가중치와 증감률을 곱한 값으로 계산된다. | 프로젝트 핵심 개념 |

## 5. 기술/프로그래밍 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **Flask** | Python으로 작성된 마이크로 웹 프레임워크. 이 프로젝트의 웹 서버 및 API 엔드포인트를 제공한다. | [Flask 공식 문서](https://flask.palletsprojects.com/) |
| **Jinja2** | Python용 템플릿 엔진. HTML에 Python 변수와 로직을 삽입하여 동적 웹 페이지를 생성한다. 이 프로젝트에서 보도자료 HTML을 렌더링하는 데 사용된다. | [Jinja2 공식 문서](https://jinja.palletsprojects.com/) |
| **pandas** | Python 데이터 분석 라이브러리. DataFrame이라는 자료구조를 사용하여 표 형태의 데이터를 효율적으로 처리한다. 엑셀 파일 읽기 및 데이터 추출에 사용된다. | [pandas 공식 문서](https://pandas.pydata.org/) |
| **openpyxl** | Python에서 Excel 파일(.xlsx)을 읽고 쓰는 라이브러리. 수식, 서식, 차트 등을 처리할 수 있다. 분석표 생성 및 수식 계산에 사용된다. | [openpyxl 공식 문서](https://openpyxl.readthedocs.io/) |
| **Blueprint** | Flask에서 애플리케이션을 모듈화하는 패턴. 이 프로젝트에서는 `routes/main.py`, `routes/api.py`, `routes/preview.py`로 라우트를 분리한다. | [Flask Blueprint 문서](https://flask.palletsprojects.com/en/latest/blueprints/) |
| **JSON** | JavaScript Object Notation의 약자. 키-값 쌍으로 구성된 경량 데이터 교환 포맷. Schema 파일과 API 응답에 사용된다. | [JSON 공식 사이트](https://www.json.org/json-ko.html) |
| **DataFrame** | pandas의 2차원 테이블 형태 자료구조. 행과 열로 구성되며, 엑셀의 시트와 유사한 개념이다. | [pandas DataFrame 문서](https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.html) |
| **KOSIS** | Korean Statistical Information Service의 약자. 국가통계포털로, 국가데이터처에서 제공하는 각종 통계 데이터를 열람할 수 있다. | [KOSIS 포털](https://kosis.kr/) |
| **KOSIS Open API** | KOSIS에서 제공하는 공개 API. 프로그래밍 방식으로 통계 데이터를 자동 수집할 수 있다. (이 프로젝트에서는 사용하지 않음) | [KOSIS Open API 가이드](https://kosis.kr/openapi/index.do) |
| **Chart.js** | JavaScript 차트 라이브러리. 이 프로젝트의 요약 페이지에서 막대+점 복합 차트를 렌더링하는 데 사용된다. | [Chart.js 공식 사이트](https://www.chartjs.org/) |
| **iframe** | Inline Frame의 약자. HTML 문서 안에 다른 HTML 문서를 삽입하는 요소. 대시보드의 미리보기 영역에서 생성된 보도자료를 표시하는 데 사용된다. | [MDN iframe 문서](https://developer.mozilla.org/ko/docs/Web/HTML/Element/iframe) |
| **fetch API** | JavaScript에서 HTTP 요청을 보내는 내장 API. 대시보드에서 서버와 비동기 통신할 때 사용한다. | [MDN Fetch API 문서](https://developer.mozilla.org/ko/docs/Web/API/Fetch_API) |
| **드래그 앤 드롭** | 마우스로 파일이나 요소를 끌어서 특정 위치에 놓는 인터랙션 방식. 대시보드의 파일 업로드 영역에서 사용된다. | [MDN Drag and Drop](https://developer.mozilla.org/ko/docs/Web/API/HTML_Drag_and_Drop_API) |

## 6. 규칙기반 시스템 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **매핑 규칙** | 엑셀의 긴 공식 명칭을 보도자료용 간결한 표현으로 변환하는 규칙. 예: "전자 부품, 컴퓨터, 영상, 음향 및 통신장비 제조업" → "반도체·전자부품" | 규칙 유형 1 |
| **계산 규칙** | 수치를 처리하는 규칙. 소수점 처리(첫째 자리까지 반올림), 백분율 변환, 천 단위 구분 등이 포함된다. | 규칙 유형 2 |
| **정렬/분류 규칙** | 데이터를 의미 있는 순서로 정렬하고 그룹화하는 규칙. 증감률 순, 지역 그룹, 상위 N개 추출 등이 포함된다. | 규칙 유형 3 |
| **조건부 텍스트 생성** | 수치의 부호나 크기에 따라 다른 표현을 사용하는 규칙. 예: 증감률 > 0 → "증가", 증감률 < 0 → "감소" | 규칙 유형 4 |
| **문장 템플릿** | 정해진 문장 구조에 데이터를 삽입하는 규칙. 예: "{지역}의 {지표}은(는) 전분기 대비 {증감률}% {증감표현}하였다." | 규칙 유형 5 |
| **결측치 처리 규칙** | 데이터가 없는 경우 안전하게 처리하는 규칙. "[ ]" 플레이스홀더 표시, 기본값 사용, 원본 값 그대로 표시 등이 포함된다. | 규칙 유형 6 |
| **의미 기반 마커** | 셀 위치(행/열 번호) 대신 텍스트 패턴으로 데이터를 찾는 방식. 예: "전국" 텍스트를 찾고, 그 오른쪽 셀에서 증감률을 읽는다. | 프로젝트 핵심 기법 |

## 7. 시스템 아키텍처 용어

| 용어 | 정의 | 비고 |

| **routes/** | Flask 라우트(API 엔드포인트)를 담은 디렉토리. `main.py`(대시보드 페이지), `api.py`(파일 업로드, 보도자료 생성), `preview.py`(미리보기)로 구성된다. | 프로젝트 디렉토리 구조 |
| **services/** | 비즈니스 로직을 담은 디렉토리. `report_generator.py`(보도자료 HTML 생성), `summary_data.py`(요약 페이지 데이터 추출), `grdp_service.py`(GRDP 데이터 처리)로 구성된다. | 프로젝트 디렉토리 구조 |
| **utils/** | 유틸리티 함수를 담은 디렉토리. `filters.py`(Jinja2 커스텀 필터), `excel_utils.py`(엑셀 파일 처리), `data_utils.py`(데이터 처리)로 구성된다. | 프로젝트 디렉토리 구조 |
| **config/** | 설정 및 상수를 담은 디렉토리. `settings.py`(경로, 시크릿 키 등), `reports.py`(보도자료 순서 및 정의)로 구성된다. | 프로젝트 디렉토리 구조 |
| **templates/** | Jinja2 템플릿 및 Generator를 담은 디렉토리. `*_template.html`(HTML 템플릿 71개), `*_generator.py`(데이터 추출 모듈 15개), `*_schema.json`(데이터 스키마 39개)로 구성된다. | 프로젝트 디렉토리 구조 |
| **DataConverter** | 기초자료 수집표를 분석표로 변환하는 클래스. `data_converter.py` 파일에 구현되어 있다. | 프로젝트 핵심 클래스 |
| **ReportGenerator** | 보도자료 HTML을 생성하는 핵심 로직. `services/report_generator.py` 파일에 구현되어 있다. | 프로젝트 핵심 클래스 |
