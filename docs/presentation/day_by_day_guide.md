# 일주일간 발표 연습 상세 가이드

## Day 1: 스크립트 전환 및 구조 파악 (1시간)

### 목표
- 전체 스크립트를 핵심 맵으로 전환
- 발표 구조 완전히 이해하기

### 상세 연습 방법

#### 1. 스크립트 분석 (20분)

**작업:**
1. 전체 발표 스크립트 작성/확인
   - `presentation_structure.md` 참고
   - 각 섹션별 내용 정리

2. 각 섹션의 핵심 메시지 파악
   - 도입부: 문제 정의, 프로젝트 소개, 목표
   - 본론 1: 아키텍처, 50개 보도자료, 대시보드
   - 본론 2: 기술적 성과, 데모
   - 결론: 성과 요약, 향후 과제, 마무리

3. 불필요한 문장 제거
   - 읽기용 문장 → 말하기용으로 변환
   - 핵심만 남기기

**체크:**
- [ ] 전체 스크립트 확인 완료
- [ ] 각 섹션 핵심 메시지 파악 완료

---

#### 2. 핵심 맵 만들기 (25분)

**작업:**
1. `key_cards_template.md` 열기
2. 슬라이드별 핵심 키워드 1-2줄 추출
   - 각 슬라이드의 핵심만 적기
   - 문장이 아닌 키워드 중심

3. 읽기용 → 말하기용 변환
   - "~하였습니다" → "~했어요"
   - 긴 문장 → 짧은 구문
   - 구어체로 변환

4. 핵심 카드 제작
   - 슬라이드별로 카드 만들기
   - 인쇄하거나 디지털로 보관

**예시:**
```
슬라이드 1: 제목
❌ 읽기용: "지역경제동향 보도자료 자동 생성 시스템에 대해 발표하겠습니다."
✅ 말하기용: "지역경제동향 보도자료 자동 생성 시스템"
            "국가데이터처 의뢰"
```

**체크:**
- [ ] 슬라이드별 핵심 카드 완성
- [ ] 읽기용 → 말하기용 변환 완료

   - 본론 1 → 본론 2: "이제 이 시스템의 기술적 성과를..."
   - 본론 2 → 결론: "마지막으로 향후 개선 과제를..."

## Day 2: 도입부 연습 (1시간)

### 목표
- 첫인상이 좋은 도입부 완성
- 자연스러운 말하기 시작

### 상세 연습 방법

#### 1. 핵심만 보고 말하기 (20분)

**작업:**
1. 도입부 핵심 카드만 보기
   - 슬라이드 1-3 핵심 카드 준비
   - 스크립트는 보지 않기

2. 스크립트 문장 그대로 말하지 않기
   - 핵심만 보고 즉석에서 설명
   - 매번 다른 표현 시도

3. 매번 다른 표현 시도
   - 1회차: "이 프로젝트는..."
   - 2회차: "저희가 개발한 시스템은..."
   - 3회차: "오늘 소개할 시스템은..."

**연습 방법:**
- 핵심 카드만 보며 3분 연습
- 문장 그대로 말하지 않기
- 자연스러운 대화 톤

**체크:**
- [ ] 핵심만 보고 말하기 가능
- [ ] 매번 다른 표현 시도

---

#### 2. 눈맞춤 연습 (20분)

**작업:**
1. 거울 앞에서 연습
   - 거울 속 자신을 청중으로 보기
   - 눈맞춤 패턴 연습

2. 슬라이드 2초 → 청중(거울) 10초
   - 슬라이드에 시선 1-2초
   - 즉시 거울(청중)로 시선 이동
   - 10초 이상 응시하며 말하기

3. Z자 스캔 패턴 연습
   - 왼쪽 끝 → 중앙 → 오른쪽 끝
   - 자연스럽게 순환

**연습 방법:**
- 거울 앞에서 도입부 전체 연습
- 슬라이드 보는 시간 최소화
- 청중 응시 시간 최대화

**체크:**
- [ ] 슬라이드 2초 이내 참고
- [ ] 청중 10초 이상 응시
- [ ] Z자 스캔 패턴 숙지

---

#### 3. 녹음 및 피드백 (20분)

**작업:**
1. 도입부 전체 녹음
   - 스마트폰으로 녹음
   - 스크립트 읽은 버전 1회
   - 자연스러운 버전 1회

2. 스크립트 읽은 버전 vs 자연스러운 버전 비교
   - 톤 차이 확인
   - 자연스러움 차이 확인
   - 개선점 파악

3. 개선점 파악
   - 불필요한 말 ("음...", "아...") 제거
   - 자연스러운 표현 더하기
   - 속도 조절

**체크:**
- [ ] 녹음 완료
- [ ] 비교 분석 완료
- [ ] 개선점 파악 완료
