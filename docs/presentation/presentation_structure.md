## 2. 본론 1: 시스템 아키텍처 및 주요 기능 (7분)

### 목표
- 시스템 구조를 쉽게 설명
- 50개 보도자료 생성 능력 강조
- 핵심 기능 명확히 전달

### 핵심 메시지

#### 2-1. 시스템 아키텍처 (2분)
- Flask 웹 애플리케이션
- 모듈화된 구조 (routes, services, utils, config)
- 입력: 분석표 엑셀, 기초자료 수집표
- 출력: HTML, HWPX, Excel, PDF

#### 2-2. 주요 기능 - 50개 보도자료 (3분)
- **요약 보도자료 9개**: 표지, 일러두기, 목차, 인포그래픽, 요약 5개
- **부문별 보도자료 10개**: 광공업생산, 서비스업생산, 소비동향, 건설동향, 수출입, 물가, 고용/실업, 인구이동
- **시도별 보도자료 18개**: 17개 시도 + GRDP 참고
- **통계표 13개**: 각 부문별 통계표 + 부록

#### 2-3. 웹 대시보드 기능 (2분)
- 드래그 앤 드롭 파일 업로드
- 실시간 미리보기
- 4개 탭 구조 (요약/부문별/시도별/통계표)
- 결측치 시각화 (노란색 형광펜)
- 다중 포맷 내보내기

## 3. 본론 2: 기술적 성과 및 데모 (7분)

### 목표
- 기술적 성과를 임팩트 있게 전달
- **규칙기반 자연어 생성**을 핵심으로 강조
- 데모 시연 자연스럽게 진행

### 핵심 메시지

#### 3-1. ⭐ 규칙기반 자연어 생성 (3분) - 가장 중요!

> **"이 시스템의 핵심은 규칙기반 자연어 생성입니다"**

**왜 규칙기반을 선택했는가?**
```
통계 보도자료 = 정확한 수치 + 정해진 서식 + 반복 패턴
→ LLM보다 규칙기반이 적합
→ 정확성 100%, 검증 가능, 일관된 결과
```

**3계층 구조 설명:**
```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  Generator  │───▶│   Schema    │───▶│  Template   │
│  (Python)   │    │   (JSON)    │    │   (Jinja2)  │
└─────────────┘    └─────────────┘    └─────────────┘
  데이터 추출        데이터 구조        텍스트 생성
  매핑 규칙          컬럼 매핑          조건부 문장
  증감률 계산        유효성 검사        자동 포맷팅
```

**핵심 규칙 예시 (발표 시 코드 보여주기):**

1️⃣ **매핑 규칙** - 업종명 표준화
```python
INDUSTRY_NAME_MAP = {
    "전자 부품, 컴퓨터, 영상, 음향 및 통신장비 제조업": "반도체·전자부품",
    # 31자 → 7자로 표준화
}
```

2️⃣ **계산 규칙** - 증감률 자동 계산
```python
growth_rate = ((current_index - prev_year_index) / prev_year_index) * 100
growth_direction = "증가" if growth_rate > 0 else "감소"
```

3️⃣ **정렬 규칙** - 증가/감소 지역 자동 분류
```python
increase_regions = sorted([r for r in regions if r["growth_rate"] > 0], 
                          key=lambda x: x["growth_rate"], reverse=True)
```

4️⃣ **텍스트 생성 규칙** - Jinja2 조건문
```html
{{ region }}의 광공업생산은 전년동분기대비 
{{ "%.1f"|format(growth_rate) }}% 
{% if growth_rate > 0 %}증가{% else %}감소{% endif %}
```

**결과 예시:**
- 입력: `{"region": "서울", "growth_rate": 3.2}`
- 출력: "서울의 광공업생산은 전년동분기대비 3.2% 증가"

**LLM과의 비교 (표로 보여주기):**

| 기준 | 규칙기반 | LLM |
|------|---------|-----|
| 정확성 | ✅ 100% | ❌ Hallucination |
| 검증가능성 | ✅ 명시적 | ❌ Black Box |
| 비용 | ✅ 무료 | ❌ API 비용 |
| 속도 | ✅ 밀리초 | ❌ 초 단위 |
