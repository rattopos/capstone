# 핵심 카드 템플릿

> **사용법**: 각 슬라이드별로 핵심 키워드 1-2줄만 적어서 말하기 연습에 활용하세요.
> 스크립트 문장 그대로 말하지 말고, 이 핵심만 보고 자연스럽게 설명하세요.

## 연습 팁

1. **각 카드를 1-2초만 보기**
   - 카드 확인 → 즉시 청중으로 시선
   - 핵심만 파악하고 말하기

2. **매번 다른 표현 시도**
   - 같은 내용이라도 표현 다르게
   - 자연스러운 대화 톤 유지

3. **연결어 활용**
   - "그래서", "이제", "다음으로"
   - 섹션 간 자연스러운 전환

4. **숫자는 자연스럽게**
   - "50개 보도자료" → "총 50개의 보도자료를"
   - "104개 커밋" → "총 104번의 커밋을"
